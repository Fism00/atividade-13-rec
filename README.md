# atividade-13-rec

Hoje vocês deverão atuar como técnicos de desenvolvimento e elaborar um planejamento técnico de um projeto fictício, como se estivessem propondo uma solução a um cliente real. O grupo pode escolher e definir o contexto (por exemplo: app de transporte, sistema de controle de estoque, automação residencial, plataforma de ensino, etc.) e apresentar como as tecnologias seriam escolhidas, justificadas e integradas, com foco nos requisitos não funcionais (desempenho, segurança, escalabilidade, usabilidade, integridade, manutenibilidade).
